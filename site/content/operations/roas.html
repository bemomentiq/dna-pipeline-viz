6.1

ebmTS — empirical-Bayes multi-bandit

MLE-estimate the hyperprior covariance Σprior from existing brand posteriors. Each new brand inherits the category-specific prior and shrinks toward the hyperprior as data accumulates. NeurIPS 2024 HierBayesUCB proves O(m log n · log n) regret vs O(m√n) for isolated bandits — a 5–10× cold-start speedup. 

# Empirical-Bayes hyperprior
means  = stack(brand_posteriors.values())
Sigma  = cov(means.T)
mu_pri = mean(means, axis=0)

# Hier-TS sample
mu_post, Sig_post = brand_posterior(
  brand_id, mu_pri, Sigma, obs[brand_id]
)
theta = mvn_sample(mu_post, Sig_post)
arm   = argmax(theta)

6.2

UCB portfolio allocator · ROAS floor

Global constraint: sum(spend) ≤ B_total and portfolio_ROAS ≥ 2.0. UCB score per brand = μ̂roas + √(2 log t / n). Sorted-greedy allocation to brands with highest UCB until budget exhausted. Brands below the ROAS floor are automatically budget-starved. 

for t in weeks:
  ucb = {b: mu[b] + sqrt(2*log(t)/n[b])
         for b in brands}
  # sorted greedy
  for b in sorted(ucb, key=ucb.get, reverse=True):
    alloc[b] = min(rem, rem * ucb[b]/sum(ucb.values()))
    rem -= alloc[b]
  # PID pacing trim
  alloc = pid_pace(alloc, target_roas=2.0)
