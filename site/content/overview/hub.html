And the build itself runs autonomously: GitHub Issues as source of truth + CC queue for fleet dispatch + PR Guardian for auto-merge healing.

3-LAYER BUILD ARCHITECTURE

Issues + CC Queue + PR Guardian

L1
GitHub Issues — bemomentiq/momentiq-dna. Milestones M10–M13. Labels dna:*, fleet:*. Human review surface + audit trail.
L2
CC Queue — Railway. dna-issue-sync cron (every 10m) transforms issues → cc_tasks with agentBriefing. Auto-dispatch to healthy Mini lane. Retry + rebalance autonomous.
L3
PR Guardian — forked for DNA repo. Scans open PRs every 15m. Classifies stuck (fix-ci, rebase, resolve-conflicts, nudge). Auto-merges on green.
