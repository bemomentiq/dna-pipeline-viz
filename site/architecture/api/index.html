API surface

All `/api/v1/dna/*` endpoints.

12 · API Catalog

Every /api/v1/dna/* endpoint

All net-new, all behind the existing auth middleware. No existing endpoints touched.

GET/api/v1/dna/profile/latest

Latest synthesized DNA profile across all categories. Used by ScriptSage at script-plan time.

Response
{
  "profile_date": "2026-04-13",
  "categories": {
    "vacuums": { "n": 22, "hook_type_weights": { … }, … },
    …
  }
}
GET/api/v1/dna/profile/{category}

Category-scoped profile including override flags and prior blending provenance.

Response
{
  "category": "vacuums",
  "n": 22,
  "median_gpm": 14.20,
  "overrides": { "duration_range": [65, 90], "demo_type": "before_after" },
  "prior_source": "own"
}
POST/api/v1/dna/generate

Create a generation job with DNA-seeded config. Delegates to queue-app dispatcher.

Request
{
  "product_id": "tts_12345",
  "category": "vacuums",
  "variant_count": 50,
  "force_arms": null
}
Response
{ "job_id": "job_01HV…", "arms": { "hook_type": "problem_pain", … } }

INTERNALPOST /dna/sample

ScriptSage-only sampling endpoint. Draws arm values for a specific (attribute, category) at job-create time.

Response
{ "attribute": "hook_type", "arm": "problem_pain", "theta": 0.62 }
INTERNALPOST /dna/drift/check

Weekly cron trigger — compares latest profile to prior run. Returns JS-divergence per attribute.

Response
{ "js_div_max": 0.09, "attributes_flagged": [], "reset_required": false }
