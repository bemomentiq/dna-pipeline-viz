## v1 — Kalodata → AI Content Platform Training Pipeline (2026-04-22)

The original 33-section walkthrough — overview, flow, phases, architecture,
configs, bandits, reward, gates, trace, API, governance, monitoring, rollout,
glossary, quickstart, schema, cron, invariants, SOTA gap, compounder, ROAS,
convergence, impl, cost/ROI, failures, integration, build plan, gameplan.

```bash
cd v1-walkthrough
python3 build.py        # rebuilds index.html from partials/
python3 -m http.server  # serve locally
```

Editing workflow:

1. Edit a partial in `v1-walkthrough/partials/` (e.g. `100-roadmap.html`).
2. Run `python3 build.py`.
3. `index.html` is regenerated by concatenating partials in lexicographic order.

**Notes:** Preserve the NBSP (`\u00a0`) inside `schema\u00a0ready`. `build.py`
is intentionally dumb — no templating, no variables; ordering comes purely from
the numeric prefix. `split.py` was a one-time bootstrap tool and should not be
re-run.
